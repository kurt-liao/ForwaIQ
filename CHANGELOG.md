# 📝 更新日誌

本系統重要更新記錄。

---

## [2.0.0] - 2025-10-20

- 🎉 資料庫核心全面從 KV Store 遷移至 PostgreSQL
- ✨ 強化 API 與資料查詢效能（查詢速度最高提升 10 倍）
- ✨ 新增：9 個主要資料表、效能優化索引、4 個視圖
- ✨ 支援複雜搜尋、批次操作、自動格式轉換
- 📚 文檔集中管理於 `docs/`，新增 API & 架構說明、遷移指南等
- 🔒 外鍵與約束保障資料一致性，預設支援 RLS
- 🗑️ 完全移除 KV Store 與相關臨時檔案
- 📈 PostgreSQL 實現資料空間節省
